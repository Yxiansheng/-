### 防抖
#### 作用：用于防止同一函数多次执行，导致上一次函数执行过程未结束，就再次执行函数出现的异常，或者由于多次执行函数，导致大量资源浪费的异常。常见例子有
#### 原理：通过使用倒计时计时器结合闭包，每次触发函数执行时，判断上次函数执行设定的计时器是否为计时完毕，若完毕则可再次设置一个定时器，同时在其回调函数执行函数；若上一计时器未计时结束，则直接退出。
```
const debounce = (fun, timeout) => {
  let timer = null;
  return function() {
    const that = this
    const args = arguments
    if (timer) clearTimeout(timer)
    timer = setTimeout(() => {
      fun.apply(that, args)
      clearTimeout(timer)
      timer = null
    }, timeout)
  }
}
```
